# Прогнозирование заказов такси
Цель работы: построить модель, предсказывающую количество заказов такси на следующий час. Входные данные: исторические данные о заказах такси в аэропортах. Ход работы: 
1. Подготовка данных;
2. Анализ данных;
3. Обучение моделей;
4. Тестирование.

### Тема:
Временные ряды

### Используемые библиотеки:
pandas, matplotlib, statsmodels, sklearn
